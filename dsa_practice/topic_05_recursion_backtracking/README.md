# Topic 05 - Recursion and Backtracking

Recursion solves a problem by solving smaller versions of the same problem. Backtracking explores choices and undoes them when a path is complete or invalid.

## Brief Theory

| Concept | Use |
|---------|-----|
| Base case | Stops recursion |
| Recursive case | Reduces problem |
| Call stack | Stores active calls |
| Backtracking | Choose, explore, undo |
| Pruning | Skip impossible paths |
