# Topic 07 - Trees

Trees are hierarchical structures. Binary trees have at most two children per node. Binary search trees keep left values smaller and right values larger.

## Brief Theory

| Concept | Use |
|---------|-----|
| Root | Top node |
| Leaf | Node with no children |
| DFS traversal | Pre/in/post order |
| Height | Longest path to leaf |
| BST | Ordered binary tree |
