# Topic 08 - Graphs

Graphs model vertices connected by edges. Use adjacency matrices for small dense graphs and adjacency lists for larger sparse graphs.

## Brief Theory

| Concept | Use |
|---------|-----|
| Vertex | Node in graph |
| Edge | Connection |
| Directed graph | One-way edge |
| BFS | Shortest path in unweighted graph |
| DFS | Reachability, components |
